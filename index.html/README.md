# Market Folder Web Project

A simple multi-page static website for a market/shop interface. This project contains HTML, CSS, and JavaScript files for a series of pages including market browsing, product details, user sign-in, password recovery, VIP info, contact form, and more.

## Project Structure

- `Market.html` / `Market.css` / `Market.js`
  - Main market page and related styles/scripts.
- `Product.html` / `Product.css` / `Product.js`
  - Product detail page.
- `Sign in.html` / `Sign in.css` / `Sing in.js`
  - User sign-in interface.
- `Forgot password.html` / `forgot password.css` / `Forgot password.js`
  - Password recovery page.
- `VIP.html` / `VIP.css` / `VIP.js`
  - VIP membership or special offers page.
- `Contact Us.html` / `Contact Us.css`
  - Contact form page.
- `Form.html` / `Form.css` / `form.js`
  - General form page.
- `Images Folder/`
  - Image assets used by the web pages.

## How to Use

1. Open the `Market Folder` in your editor.
2. Open `Market.html` in a web browser to view the main page.
3. Navigate between pages using the site's links or open any HTML file directly in the browser.

## Notes

- All pages are static and do not require a web server to run.
- Make sure the `Images Folder` stays in the same location relative to the HTML files if the pages use local image assets.
- If JavaScript behavior is not working, open the browser console to check for errors.

## Recommendation

Start with `Market.html` as the landing page, then explore the other pages such as `Product.html`, `Sign in.html`, `VIP.html`, and `Contact Us.html`.
